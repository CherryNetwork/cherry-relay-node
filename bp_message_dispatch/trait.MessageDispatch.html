fn dispatch<P: FnOnce(&AccountId, Weight) -> Result<(), ()>>(
    source_chain: ChainId,
    target_chain: ChainId,

    message: Result<Self::Message, ()>,
    pay_dispatch_fee: P
) -> MessageDispatchResult

Dispatches the message internally.

source_chain indicates the chain where the message came from. target_chain indicates the chain where message dispatch happens.

id is a short unique identifier of the message.

If message is Ok, then it should be dispatched. If it is Err, then it’s just a sign that some other component has rejected the message even before it has reached dispatch method (right now this may only be caused if we fail to decode the whole message).

Returns unspent dispatch weight.
