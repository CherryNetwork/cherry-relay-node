Expand description

Check if the message is allowed to be dispatched on the target chain given the sender’s origin on the source chain.

For example, if a message is sent from a “regular” account on the source chain it will not be allowed to be dispatched as Root on the target chain. This is a useful check to do on the source chain before sending a message whose dispatch will be rejected on the target chain.
